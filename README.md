<h2>⚙️ 주요 기능</h2>

**1.객실 정보 제공**
* 객실별 사진, 설명, 편의시설, 최대 인원 표시

**2. 예약 시스템**
* FullCalendar 5.11.3 사용
  * 드래그 방식으로 연박 예약 지원 (날짜 선택 시 하이라이트 표시)
  * 예약 가능 여부 시각적 표시
  * 객실별 예약 내역 반영

**3.고객 편의 기능**
* 카카오맵 API 지도를 사용하여 펜션의 정확한 위치 안내
* 예약 내역 조회 및 취소 기능
